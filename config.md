# Config

## API Keys

Set these as environment variables so scripts can read them without hardcoding secrets.

### How to set permanently on Mac/Linux
Add to your `~/.zshrc` or `~/.bashrc`:
```bash
export PHISHNET_API_KEY="your_key_here"
export SETLISTFM_API_KEY="your_key_here"   # optional
```
Then run: `source ~/.zshrc`

### Get your keys
- **phish.net API key** (free): https://api.phish.net/keys/ — instant, no approval needed
- **setlist.fm API key** (free): https://www.setlist.fm/settings/api — for Dead/non-Phish artists
